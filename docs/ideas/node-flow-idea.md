# ノードベース可視化システムのアイディア

## 概要

-   ノード間の通信を粒子で表現し、川の流れのようなビジュアライゼーションを目指す。
-   ローカルネットワーク（右側）とインターネット（左側）を分けて表示。

## 参考画像

![スケッチ](images/20251126_182356.JPG)

## 実装方針

-   Boids アルゴリズムを使用して粒子の動きを自然に。
-   ノードの自動生成と動的配置。
-   パーティクルの軌道は描画せず、粒子のみを可視化。

## 今後の課題

-   粒子の動きの最適化。
-   ノード間の通信量に応じた流れの太さの表現。
-   スケッチ画像の追加。
